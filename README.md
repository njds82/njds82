## ⭐ المشاريع المميزة

### [Salatk](https://github.com/njds82/salatk)
تطبيق ويب PWA متقدم **لإدارة حياة المسلم** — يجمع بين مواقيت الصلاة، متتبع العادات، إدارة المهام، ألعاب إسلامية تعليمية، لوحة صدارة تنافسية، ومتجر سمات.

**المميزات البارزة:**
- نظام نقاط ومستويات متقدم مع إحصائيات تفصيلية
- تحديات تعليمية إسلامية متعددة المراحل
- دعم اللغة العربية والتقويم الهجري
- مصادقة آمنة عبر Supabase مع إدارة صلاحيات متقدمة
- مزامنة فورية عبر Realtime و IndexedDB

---

### [NIO](https://github.com/njds82/NIO-Num_In_One-)
نظام **تشفيري موزّع** لمزامنة الملفات والمجلدات بين أجهزة Arch بأمان عالي، باستخدام هاتف Android كوسيط وسيط.

**التقنيات**: Rust • Cryptography • Android JNI • Linux daemons

---

### [A-hyprland](https://github.com/njds82/A-hyprland)
مجموعة **إعدادات احترافية لـ Hyprland** على Arch Linux — نافذة manager خفيفة سريعة مع شريط حالة متقدم، معرّف جزئي، ومشغل تطبيقات.

**المميزات البارزة:**
- تكوين Hyprland محسّن للأداء
- +50 اختصار لوحة مفاتيح مخصص (بدعم AR/EN)
- ثيمات وخلفيات احترافية
- سكريبت تثبيت آلي مع النسخ الاحتياطي
- دعم كامل للصوت والإضاءة والعروض

**التقنيات**: Shell/Bash • Hyprland • Waybar • Linux config

---

### [Rorocode](https://github.com/njds82/rorocode)
أداة CLI قوية لـ Linux تجمّع **نماذج AI من عدة APIs** لتشكيل **فريق من الوكلاء الذكيين**، مما يقلل التكاليف واستهلاك tokens ويرفع الكفاءة.

**المميزات البارزة:**
- دعم وكلاء متعددين يعملان معاً
- تقليل تكاليف API بذكاء
- تكوين مرن وسهل الاستخدام
- تتبع استهلاك tokens والأداء

**التقنيات**: Makefile/Scripting • AI/LLM Integration • CLI design

---

### [Phonix2you](https://github.com/njds82/phonix2you)
تطبيق **شبكة اجتماعية خاصة جداً** تركز على الخصوصية والأمان.

**التقنيات**: JavaScript
